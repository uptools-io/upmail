Professional SMTP solution for WordPress using upTools API. Send emails reliably with advanced logging and tracking capabilities.

== Description ==

upMail is a professional SMTP solution that ensures reliable email delivery from your WordPress site using the upTools API service.

= Key Features =

* Easy setup with API key
* Email delivery via upTools SMTP servers
* Comprehensive email logging
* Real-time email tracking
* Email status monitoring
* Test email functionality
* Modern, user-friendly interface
* Detailed statistics and analytics
* Secure API key storage

= Email Logging =

* Track all sent emails
* View email content and details
* Filter by status (sent/failed)
* Search functionality
* Date range filtering
* Resend failed emails
* Bulk actions support

= Statistics & Analytics =

* Overview dashboard
* Monthly statistics
* Success/failure rates
* Detailed error tracking
* Visual data representation
* Export capabilities

= Security =

* Encrypted API key storage
* Role-based access control
* Secure email transmission

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/upmail` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to upTools > upMail in the admin menu.
4. Enter your upTools API key and configure the settings.
5. Send a test email to verify the setup.

== Frequently Asked Questions ==

= Do I need an upTools account? =

Yes, you need an upTools account and API key to use this plugin. Visit uptools.io to create an account.

= Is the API key stored securely? =

Yes, the API key is encrypted before being stored in the database using WordPress security standards.

= Can I track bounced emails? =

Yes, the plugin provides detailed tracking of email delivery status, including bounces and failures.

= What happens to failed emails? =

Failed emails are logged with detailed error information and can be resent directly from the logs page.

= Can I customize the From email and name? =

Yes, you can set and force custom From email and name in the plugin settings.
